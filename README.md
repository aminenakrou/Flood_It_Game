# 🎨 Flood It Game

> Jeu de réflexion interactif développé en **HTML**, **CSS** et **JavaScript**, avec interface responsive, génération aléatoire de grille et mode défi.

## ✨ Aperçu

**Flood It** est un jeu de stratégie dans lequel le but est de remplir toute la grille avec une seule couleur en un minimum de coups.

Le joueur peut :
- choisir la taille de la grille ;
- définir le nombre de couleurs ;
- activer un mode défi ;
- jouer directement dans le navigateur.

## 🚀 Fonctionnalités

- ✅ Interface web simple et intuitive
- ✅ Développement en HTML / CSS / JavaScript
- ✅ Utilisation de Bootstrap pour la mise en page
- ✅ Génération aléatoire de la grille
- ✅ Palette de couleurs interactive
- ✅ Sélection de la couleur du prochain coup
- ✅ Mise à jour du nombre de coups joués
- ✅ Détection automatique de la victoire
- ✅ Mode défi avec limite de coups
- ✅ Fenêtre d’information intégrée

## 🧠 Règles du jeu

Le principe est simple :

- une grille colorée est générée aléatoirement ;
- le joueur choisit une couleur ;
- la zone connectée est remplie avec cette nouvelle couleur ;
- l’objectif est d’obtenir une grille entière d’une seule couleur.

En **mode défi**, le joueur doit réussir en moins de `2L` coups, où `L` représente la taille de la grille.
